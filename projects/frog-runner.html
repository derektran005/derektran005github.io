Projects | Frog Runner Deluxe

(Python/PixelPad)

Project Overview

Frog Runner Deluxe is a gravity-based 2D platformer developed over approximately seven months. The project focuses on physics-driven movement, environmental interaction, combat mechanics, and enemy behavior within a structured level layout.

Project Goals

The primary goal of this project was to design a cohesive platformer system that integrates movement physics, collision handling, camera control, and multiple combat mechanics while maintaining consistent player control and responsiveness.

Engineering Challenges

One major challenge was maintaining stable collision behavior across static platforms, moving platforms, destructible objects, and collisions between objects with differentt velocities. This required separating collision resolution logic from movement updates to avoid clipping and inconsistent player positioning.

Core Systems

Movement & Physics

  • Gravity-based vertical movement with jump control
  • Wall and floor collision handling
  • Dash and dash-attack mechanics with knockback
  • Collision with objects with high velocities bounces entities

Camera System

  • Smooth camera interpolation following player movement
  • Camera leading behavior when aiming ranged weapons

Combat Weapons

  • Melee system with directional sword slashes
  • Ranged projectile system for bow attacks
  • Physics-based interaction system for magic wand (entity pickup and throwing using mouse velocity)

Environment

  • Static and moving platforms
  • Destructible environmental objects
  • Hazard systems (spike traps)

Enemies & Bosses

  • Health and velocity systems
  • Enemy patrol and attack logic
  • Boss encounter with multiple attack patterns and state transitions

This project reflects a long-term iterative development process, where new mechanics were layered onto existing systems while preserving stability and player control.
